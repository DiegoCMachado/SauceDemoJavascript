
# Saucedemo

* Projeto criado para automatizar um fluxo de compras no site SauceDemo.
* Estruturado em formato Gherkin sem a necessidade de utilização do Cucumber

**Tecnologias Utilizadas**
* Javascript
* VSCode
* Cypress
* Faker
* GitHub Workflows
